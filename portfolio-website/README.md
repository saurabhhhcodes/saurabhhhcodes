## Local Development

```bash
npm install
npm start
```

The app runs on `http://localhost:3000`.

## Production Build

```bash
npm run build
```

The optimized static output is written to `build/`.

## Deploy

```bash
npm run deploy
```

This publishes the built site to the `saurabhhhcodes/saurabhhhcodes` GitHub Pages repository target configured in `package.json`.
